Research Project | Robotics and Automation Lab | Nov 2023 - Present

Project Overview

This research project focuses on developing an advanced framework for robotic manipulation by integrating Deep Reinforcement Learning (DRL) with Large Language Models (LLMs). The primary objective is to enable robots to perform complex tasks in unstructured environments with improved adaptability and enhanced human-robot interaction capabilities.

Key Features & Innovations

HiPPO-DIK Framework

Hierarchical Proximal Policy Optimization with Differentiable Inverse Kinematics for real-time joint configuration in high-DOF robotic manipulators.

Natural Language Integration

Seamless integration of Large Language Models for intuitive human-robot interaction through natural language commands.

Real-time Adaptation

Dynamic environment adaptation capabilities allowing robots to handle unstructured and changing environments.

Multi-modal Learning

Integration of visual, textual, and proprioceptive data for comprehensive scene understanding and task execution.

Technical Implementation

1
Architecture Design
