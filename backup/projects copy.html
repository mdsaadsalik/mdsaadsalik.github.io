Advanced Robotic Manipulation Using DRL and LLMs

Research Project | Robotics and Automation Lab | Nov 2023 - Present

Project Overview

This research project focuses on developing an advanced framework for robotic manipulation by integrating Deep Reinforcement Learning (DRL) with Large Language Models (LLMs). The primary objective is to enable robots to perform complex tasks in unstructured environments with improved adaptability and enhanced human-robot interaction capabilities.

Technical Implementation

  • HiPPO-DIK Framework: Hierarchical Proximal Policy Optimization with Differentiable Inverse Kinematics for real-time joint configuration in high-DOF robotic manipulators
  • ROS Integration: Complete implementation within Robot Operating System framework for modularity and scalability
  • Gazebo Simulation: Comprehensive testing in simulated environments for validation and performance analysis
  • LLM Integration: Natural language processing capabilities for intuitive human-robot interaction

Research Impact

This work represents a significant advancement in robotic manipulation, bridging the gap between traditional control methods and modern AI techniques. The integration of LLMs allows for more natural and intuitive control interfaces, while DRL provides the learning capability for complex task execution.

3-DOF Articulated Robotic Arm for Industrial Applications

Academic Project | M-Tech Machine Design | 2023

Project Overview

This academic project involved the comprehensive design, simulation, and optimization of a 3-DOF articulated robotic arm specifically tailored for industrial applications. The focus was on achieving high precision, operational speed, and reliability in manufacturing environments through advanced kinematic modeling and control strategies.

Technical Implementation

  • Kinematic Analysis: Developed complete forward and inverse kinematic models for precise motion planning
  • Dynamic Modeling: Created detailed dynamic models using MATLAB Simulink to simulate real-world performance
  • Control System: Implemented advanced torque control strategies for accurate end-effector positioning
  • Structural Design: Optimized mechanical design for strength-to-weight ratio and manufacturing feasibility

Performance Metrics

The robotic arm demonstrated exceptional performance with positioning accuracy within ±0.1mm, repeatability of 99.8%, and operational speeds suitable for high-throughput industrial applications. The torque control implementation ensured smooth operation and reduced mechanical stress on components.
